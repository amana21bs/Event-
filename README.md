# 🎉 Event Webpage

A simple event webpage created using HTML as a beginner project. This project was built to practice basic web development concepts and improve my understanding of HTML structure and webpage design.

## 📌 Project Overview

This webpage includes essential information related to an event such as:

- Event title
- Date and time
- Venue details
- Event description
- Registration or contact details
- Basic webpage structure using HTML

## 🛠 Technologies Used

- HTML5

## 🎯 Purpose of This Project

The purpose of creating this project was to:

- Practice HTML fundamentals
- Learn webpage structure and layout
- Gain hands-on experience with creating and publishing websites
- Improve beginner web development skills

## 📚 What I Learned

Through this project, I learned:

- Basic HTML tags and elements
- Creating headings, paragraphs, lists, and links
- Structuring a webpage properly
- Publishing a website using GitHub Pages
